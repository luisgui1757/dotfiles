# AGENTS.md

This repository's full coding-agent guide is [CLAUDE.md](./CLAUDE.md). It is
the single source of truth for non-negotiable invariants, common workflows, test
commands, conventions, and durable rationale.

Do not duplicate that content here. This file exists so non-Claude agents have a
standard entry point; read `CLAUDE.md` before changing anything, then follow its
"When you're about to make a change" checklist. `README.md` is the human-facing
install and usage guide.
