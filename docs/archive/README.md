# Archived migration plans

This directory holds historical implementation plans from the dotfiles to
chezmoi migration. They are kept as implementation references, not living
status documents.

Use `docs/MIGRATION_STATUS.md` for current migration state, owner sign-off
caveats, uninstall behavior, and Wave C readiness tracking.

`WINDOWS_THEME_FIXES.md` is archived here because the portable Windows Terminal
seed/merge, psmux Rose Pine, Hack Nerd Font failure recording, font-change
notification, tests, and docs landed; the root-level handoff is no longer live
work.
